# My Jigsaw Puzzle

My Jigsaw Puzzle の紹介用 GitHub Pages です。

- 公開 URL: https://osobaudonmen.github.io/my-jigsaw-puzzle/
- このリポジトリは紹介ページ専用です。

## ローカル確認

GitHub Pages (Jekyll) と同等の確認をする場合は次を実行します。

```sh
bundle install
bundle exec jekyll serve
```

ブラウザで `http://localhost:4000/my-jigsaw-puzzle/` を開いて確認できます。
